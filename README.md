# Zygne-Architecture



How to use:

Step 1. Add JitPack repository to your build file
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  
  Step 2. Add the dependency
  ```
  dependencies {
	        implementation 'com.github.bardurt:Zygne-Architecture:1.0.0'
	}
  ```
